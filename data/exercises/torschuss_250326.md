## Setup

## Ablauf
1. *1* dribbel mit Ball in den Halbkreis Schuss -> Sprint zurück -> Pass auf *2* -> 1v1 gegen *2*
2. *2* Anspiel an *3* -> 2. Ball Schuss -> 1v1 gegen *1* auf das kleine Tor
3. *3* Driblling durch Hüttchentor -> drehen 1v1 gegen *4* -> 1v1 verteidigen gegen *4*
4. 1v1 gegen *3* sobald *3* durch Hüttchentor gedribbelt ist -> Nach pass von 5 1v1 gegen *3*
5. *5* Pass auf *4* -> Doppelpass auf *6* -> Sprint Grundlinie -> mit Ball Doppelpass mit *6* -> dribbling um das Hüttchen 1v1 gegen *6* 
6. *6* Doppelpass mit *5* Dribbling um das Hüttchen Schuss -> Doppelpass mit *5* -> 1v1 gegen *5*


## Coaching points
- Höchstmögliches Tempo 
- enge Ballführung 
- Verteidigen bis Ball im Tor (Ende)
